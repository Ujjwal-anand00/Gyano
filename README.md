# Gyano – Digital Learning Platform

Gyano is a full-stack digital learning platform where teachers can create courses and students can enroll, learn lessons, and track their progress.

## 🚀 Features

### Student
- User registration and login
- Browse available courses
- Enroll in courses
- Watch lessons
- Track learning progress
- Dashboard with enrolled courses

### Teacher
- Create and manage courses
- Add lessons to courses
- Track student progress
- View student details

### Platform
- Authentication using JWT
- Course management
- Lesson player
- Progress tracking
- Role-based access (Student / Teacher)

## ⚙️ Installation

### 1️⃣ Clone the repository

```
https://github.com/Ujjwal-anand00/Gyano-Digital-Learning-Platform
```

### 2️⃣ Install dependencies

Frontend
```
cd client
npm install
```
Backend
```
cd server
npm install
```

---

### 3️⃣ Run the project

Backend
```
cd server
node server.js || npm run dev
```
Frontend
```
cd client
npm run dev
```
